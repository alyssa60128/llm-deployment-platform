# LLM Deployment and Observability Platform

A learning-oriented platform for deploying simulated LLM workloads and
observing them through metrics, logs, and traces.

## Version 1 goals

- Build a FastAPI control plane
- Simulate vLLM-compatible inference workloads
- Practice automated testing and CI/CD
- Export metrics to Prometheus and Grafana
- Export structured logs to Loki
- Export traces with OpenTelemetry and Tempo

## Planned later

- Kubernetes deployment
- Terraform infrastructure
- AWS deployment
- Real vLLM GPU runtime
